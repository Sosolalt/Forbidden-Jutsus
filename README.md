


### Sync on Mobile

1) Download [Termux](https://play.google.com/store/apps/details?id=com.termux&hl=fr&gl=US)
2) ```pkg update && pkg upgrade```
3) ```pkg install git```
4) ```pkg install gh```
5) ```termux setup storage```
6) ```gh auth login```
7) git clone and have fun
